# pdmenu-gpg
pdmenu-gpg

Use pdmenu to execute common gpg commands. 

* gpg key verification
* gpg key generation
* etc

TEST

You can test the pdmenurc with:

$pdmenu pdmenurc 

INSTALL

If you like it, you can move it to ~/.pdmenurc 

RECOMMENDED PROGRAM DEPENDENCIES

rng-tools (for generating keys)
gnupg-curl 
hopenpgp-tools
